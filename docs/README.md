# Nunki — Documentation

Documentation for developers who want to use, contribute to, or extend Nunki.

**Submissions:** Public forms and CSV column reference live in **[COMMUNITY_FORMS.md](COMMUNITY_FORMS.md)**. The **`contributions/`** and **`tools/`** trees (queue, merged JSON, local UI) are **not in this repo** — see the callout there and in the root **README** so agents don’t expect those paths in a clone.

- **[DATA_PIPELINE.md](DATA_PIPELINE.md)** — Fetch vs `apply-merged`, deploy (`build:deploy`), what overwrites what
- **[COMMUNITY_FORMS.md](COMMUNITY_FORMS.md)** — Google Form links, CSV import, column maps
